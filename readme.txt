Instalation Instructions
1.Open database_conn.php in an apropriate text editor and change the values of the following variables:
	$host
	$username	
	$password
to the coresponding values for your database server.

2.Copy all files and folders onto your web server. 
3.Run the EventsSystem.Sql script to install the database.
